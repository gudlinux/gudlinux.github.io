GUD Linux

What is GUD Linux?

 GUD Linux is a distribution for amd64 (x86_64) computers. It isn't multilib, which means that it's not possible neither to run 32bit softwares nor to compile them which makes the distro fully compatible with amd64 processors.
It doesn't use any package manager, yet, which means that you have to install everything from sources

Why GUD Linux?

 GUD Linux is entirely built from scratch and tries to follow LSB, FHS and POSIX standardsas as much as possible. It also contains an Xorg environment with lightdm as DM, and i3wm as WM, which makes it very light and compatible also with oldest computers, but which can run powerful softwares like firefox, gimp et cetera.

Where can I download GUD Linux?

 GUD Linux can be downloaded here:
_Bootable Live Disk: https://sourceforge.net/projects/gudlinux-live
_Tarball (can also be downloaded after booted the Live Disk): https://sourceforge.net/projects/gudlinux-tar

How to install GUD Linux?

 After booted GUD Linux Live Disk, you will find a file called "install.txt"; simply open it with nano or vim editor and follow the instructions!
Otherwise, instructions can also be found HERE!
